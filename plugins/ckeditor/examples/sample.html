CKeditor - Sample use of Javascript Virtual Keyboard plugin

This page shows VirtualKeyboard plugin usage for the CK rich-text editor.

    Known issues
  • Plugin works in the dialog mode only
  • Virtual "Del" and "BackSpace" keys does not delete elements: images, tables, paragraphs, etc
  • Deletion does not take in count continuos whitespaces, like real backspace does
  • Plugin has only partial Opera support because of its weird selection implementation and incorrect key events processing
