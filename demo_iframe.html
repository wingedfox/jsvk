    How to use this demo:
  1. Click the 'keyboard' button below
  2. Click virtual buttons with mouse and the the text in the input field
  3. Focus another field to tell keyboard to use it as input target
  4. Enter some more text with mouse, to check that keyboard switched field successfully
    5. Repeat steps 2-4, using keyboard, to check on-the-fly keys translation
